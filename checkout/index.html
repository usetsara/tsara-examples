Tsara Inline Checkout

This example opens Tsara checkout with the inline script. Replace the placeholder values with your real public key, customer data, and redirect URLs for success, cancel, and timeout handling.

Sample payload

{
  "public_key": "pk_live_xxxxx",
  "trx_id": "ORDER-1234567890",
  "amount": 1000,
  "currency": "NGN",
  "email": "customer@example.com",
  "phone": "08012345678",
  "name": "Test Client",
  "redirect_url": "https://yourwebsite.com/checkout/return",
  "success_url": "https://yourwebsite.com/checkout/success",
  "cancel_url": "https://yourwebsite.com/checkout/cancelled",
  "timeout_url": "https://yourwebsite.com/checkout/expired"
}
